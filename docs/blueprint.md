# **App Name**: libribooks.com

## Core Features:

- Book Review Display: Show book reviews with titles, summaries, and external links to purchase.
- Book Categorization: Organize books into categories and tags.
- AI Review Summarization: Use an AI tool to generate a concise summary of each book review, enhancing search engine discoverability by offering a summarized version of the page's main content to search engines. The LLM will reason about when or if the summarization helps discovery.
- Search Engine Optimization: Automate generation of static pages requested by search engines such as Google, optimized for SEO.
- Link to Purchase: Include a clear and direct link to the book seller's site for each reviewed book.
- Dynamic Effects and Animations: Subtle animations to create a professional and engaging user experience.

## Style Guidelines:

- Primary color: Muted blue (#5D9EBC), evoking trust and knowledge.
- Background color: Light gray (#F0F2F5), for a clean and modern look.
- Accent color: Soft coral (#E9967A), used to highlight key interactive elements.
- Body font: 'Literata', a transitional serif with a literary, vintage, slightly formal feel, for body text.
- Headline font: 'Space Grotesk', A proportional sans-serif with a computerized, techy, scientific feel, suitable for headlines; use 'Literata' for body text.
- Simple, professional icons to represent book categories and actions.
- Clean and structured layout with clear content hierarchy for easy navigation.
- Subtle transitions and hover effects to enhance user engagement.
